# Support

If you encounter a problem with this mod, please provide the following information:

1. **Describe the problem clearly** – what happens, and what you expected to happen.
2. **Steps to reproduce the issue** – list each step in detail so the bug can be recreated. Include opening the game, enabling the mod, and the exact actions that trigger the issue.
3. **Environment information** – such as your platform (Windows, Android, macOS, iOS), Geometry Dash version, and device model if applicable.
4. **Other mods installed** – list any other mods that might affect the game's behavior.
5. **Screenshots, logs, or files** – attach relevant screenshots, logs, or `.geode` files to help diagnose the issue.
6. **Mod version** – specify which version of Classic Portals you are using.
7. **Date of the report** – to indicate whether the issue is recent or has been ongoing.
8. **Additional notes** – any other details that might help reproduce or understand the problem.

Providing this information ensures the issue can be understood and addressed efficiently.
