# 1.0.0
- First release of Classic Portals.
- Restored the classic portal style from Geometry Dash 1.0 up to 2.2 release date.
- Hidden modern game mode icons added after 2.2.
- Removed modern glows, keeping the portal ring simple and clean.
- Works with all current portals: Cube, Ship, Ball, UFO, Wave, Robot, Spider, Swing, Gravity, Dual, Mirror.
- Compatible with Geometry Dash v2.2074 on Windows, Android, Mac, and iOS.
- Requires Geode v4.10.0 and geode.node-ids >= v1.20.0.
