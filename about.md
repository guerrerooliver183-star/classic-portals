# Classic Portals

**Author:** Oliver Guerrero  
**Version:** 1.0.0  
**Compatibility:** Geometry Dash v2.2074 (Windows, Android, Mac, iOS)  
**Geode Version:** 4.10.0  
**Dependencies:** geode.node-ids >= v1.20.0  

**Description:**  
Renders the portals in a style similar to the classic look, restoring a simpler and cleaner appearance.  
This mod hides the modern game mode icons and glows, keeping the portal ring simple as seen from versions 1.0 up to the 2.2 release date.  

**Supported Portals:**  
Cube, Ship, Ball, UFO, Wave, Robot, Spider, Swing, Gravity, Dual, Mirror

Relive the classic portal style while playing the modern version of Geometry Dash!
